# ElainaInSpire
突发奇想想给伊蕾娜写一个尖塔mod

由于原作的战力表现不突出，卡牌设计就以“记录日记”和“旅行”为主题了。机制介绍、部分卡牌和遗物设计如Elaina.xlsx所示。

代码苦手，还在学，欢迎各路大佬指点。
